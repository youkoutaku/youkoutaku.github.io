フィルタ設計の基礎

信号処理におけるフィルタとは、信号の周波数特性を操作するものであり、信号の不要な周波数成分を除去したり減衰させたりして、必要な周波数成分のみからなる信号を得る処理を行うものです。

フィルタの種類​

基本的なフィルタタイプ​

  1. LPF（Low-Pass Filter、低域通過フィルタ）

    • 低い周波数成分を通し、高い周波数成分を遮断
    • 用途：ノイズ除去、アンチエイリアシング

  2. HPF（High-Pass Filter、高域通過フィルタ）

    • 高い周波数成分を通し、低い周波数成分を遮断
    • 用途：DC成分除去、高周波ノイズ強調

  3. BPF（Band-Pass Filter、帯域通過フィルタ）

    • 特定の周波数帯域のみを通す
    • 用途：特定周波数成分の抽出、通信システム

  4. BEF（Band-Elimination Filter、帯域阻止フィルタ）

    • 特定の周波数帯域を遮断
    • 用途：ノッチフィルタ、電源周波数ノイズ除去

理想的なフィルタの条件​

理想的なフィルタは以下の特性を満たします：

  1. 通過域で利得（ゲイン）が一定
  2. 過渡域での傾きが急峻
  3. 阻止域の利得が0
  4. 直線位相特性（通過域で位相特性が周波数に比例して直線的に変化）
理想と現実

実際のフィルタでは、これらすべての条件を同時に満たすことは不可能であり、用途に応じて最適化する特性を選択する必要があります。

アナログフィルタ設計​

フィルタ設計のステップ​

  1. フィルタの近似：所望の特性をもつ伝達関数を設計
  2. 回路実現：求められた伝達関数を抵抗、コンデンサ、コイル、オペアンプなどの素子を用いて回路として実現
設計の基本方針

フィルタの設計は**LPF（低域通過フィルタ）**が基本となっており、他のフィルタタイプは周波数変換により実現されます。

代表的なフィルタ​

  1. 一次遅れ系によるフィルタ
  2. バターワースフィルタ（Butterworth filter）
  3. チェビシェフフィルタ（Chebyshev filter）

バターワースフィルタ​

周波数変換​

実際にフィルタを設計する際には、LPFを基本として、他のフィルタタイプに変換できます。

遮断周波数の変換（LPF）​

s→sω1s \rightarrow \frac{s}{\omega_1}s→ω1​s​

通過域：[0,ω1][0, \omega_1][0,ω1​]、ω1\omega_1ω1​：LPFの遮断周波数

HPFへの変換​

s→ωhss \rightarrow \frac{\omega_h}{s}s→sωh​​

通過域：[ωh,∞)[\omega_h, \infty)[ωh​,∞)、ωh\omega_hωh​：HPFの遮断周波数

BPFへの変換​

s→s2+ω1ω2Bωss \rightarrow \frac{s^2 + \omega_1\omega_2}{B_\omega s}s→Bω​ss2+ω1​ω2​​

通過域：[ω1,ω2][\omega_1, \omega_2][ω1​,ω2​]、BωB_\omegaBω​：帯域幅

ここで：

  • ω1,ω2\omega_1, \omega_2ω1​,ω2​：下側・上側遮断周波数
  • Bω=ω2−ω1B_\omega = \omega_2 - \omega_1Bω​=ω2​−ω1​：帯域幅

BEFへの変換​

s→Bωss2+ω1ω2s \rightarrow \frac{B_\omega s}{s^2 + \omega_1\omega_2}s→s2+ω1​ω2​Bω​s​

阻止域：[ω1,ω2][\omega_1, \omega_2][ω1​,ω2​]、BωB_\omegaBω​：帯域幅

変換の物理的意味

これらの変換は、複素周波数平面上での幾何学的変換に対応しており、元のLPFの特性を保持しながら所望の周波数特性を実現します。

フィルタ設計の実際​

設計パラメータ​

  1. 遮断周波数：ωc\omega_cωc​
  2. 次数：NNN（急峻性とリップルのトレードオフ）
  3. 通過域リップル：許容できる変動
  4. 阻止域減衰量：必要な減衰レベル

性能評価指標​

  • 通過域リップル：通過域での利得変動
  • 阻止域減衰量：不要信号の抑圧レベル
  • 過渡域幅：通過域から阻止域への遷移幅
  • 群遅延特性：位相歪みの評価

まとめ​

フィルタ設計は信号処理において基礎的かつ重要な技術です：

  1. 基本理解：各フィルタタイプの特性と用途

  3. 周波数変換：LPFから他のフィルタタイプへの変換技術
  4. 実用性：設計パラメータと性能指標の関係
