Dynamic Programming for Discrete-Time Systems

This document presents the mathematical framework of dynamic programming for discrete-time optimal control problems. We derive the recursive Bellman equations that form the foundation of optimal control theory.

Discrete-Time System Formulation​

System Dynamics​

Consider a general discrete-time system:

xk+1=f(xk,uk)x_{k+1} = f(x_k, u_k)xk+1​=f(xk​,uk​)

Where:

  • xk∈Rnx_k \in \mathbb{R}^nxk​∈Rn: system state at time kkk
  • uk∈Rmu_k \in \mathbb{R}^muk​∈Rm: control input at time kkk
  • f(⋅,⋅)f(\cdot, \cdot)f(⋅,⋅): system dynamics (linear or nonlinear)

Performance Function​

The general optimal control objective is to minimize:

J=h(xN)+∑k=0N−1g(xk,uk)J = h(x_N) + \sum_{k=0}^{N-1} g(x_k, u_k)J=h(xN​)+k=0∑N−1​g(xk​,uk​)

Where:

  • h(xN)h(x_N)h(xN​): terminal cost function
  • g(xk,uk)g(x_k, u_k)g(xk​,uk​): stage cost function
  • x0x_0x0​: initial state (given)
  • xNx_NxN​: terminal state
  • NNN: finite time horizon

Control Task​

Find the optimal control sequence:

u0∗,u1∗,…,uN−1∗u_0^*, u_1^*, \ldots, u_{N-1}^*u0∗​,u1∗​,…,uN−1∗​

that minimizes the performance function JJJ subject to the system dynamics and constraints.

Backward Multi-Stage Dynamic Programming​

The key insight of dynamic programming is to solve the problem backwards in time, starting from the final stage and working towards the initial time.

Stage N: Terminal Stage (k=Nk = Nk=N)​

At the terminal stage, there are no more control decisions to make:

JN→N(xN)=h(xN)J_{N \to N}(x_N) = h(x_N)JN→N​(xN​)=h(xN​)

Since this is the final stage:

JN→N∗(xN)=h(xN)J_{N \to N}^*(x_N) = h(x_N)JN→N∗​(xN​)=h(xN​)

The optimal cost-to-go from the terminal stage is simply the terminal cost.

Stage N-1: One Step to Go (k=N−1→Nk = N-1 \to Nk=N−1→N)​

For the second-to-last stage, we consider both the immediate cost and the terminal cost:

JN−1→N(xN,xN−1,uN−1)=h(xN)+∑k=N−1N−1g(xk,uk)=JN→N(xN)+g(xN−1,uN−1)\begin{aligned} J_{N-1 \to N}(x_N, x_{N-1}, u_{N-1}) &= h(x_N) + \sum_{k=N-1}^{N-1} g(x_k, u_k) \\ &= J_{N \to N}(x_N) + g(x_{N-1}, u_{N-1}) \end{aligned}JN−1→N​(xN​,xN−1​,uN−1​)​=h(xN​)+k=N−1∑N−1​g(xk​,uk​)=JN→N​(xN​)+g(xN−1​,uN−1​)​

Since xN=f(xN−1,uN−1)x_N = f(x_{N-1}, u_{N-1})xN​=f(xN−1​,uN−1​), we can express the cost in terms of the decision variables:

JN−1→N(xN−1,uN−1)=JN→N∗(f(xN−1,uN−1))+g(xN−1,uN−1)J_{N-1 \to N}(x_{N-1}, u_{N-1}) = J_{N \to N}^*(f(x_{N-1}, u_{N-1})) + g(x_{N-1}, u_{N-1})JN−1→N​(xN−1​,uN−1​)=JN→N∗​(f(xN−1​,uN−1​))+g(xN−1​,uN−1​)

Optimal cost-to-go:

JN−1→N∗(xN−1)=min⁡uN−1[JN→N∗(f(xN−1,uN−1))+g(xN−1,uN−1)]J_{N-1 \to N}^*(x_{N-1}) = \min_{u_{N-1}} \left[ J_{N \to N}^*(f(x_{N-1}, u_{N-1})) + g(x_{N-1}, u_{N-1}) \right]JN−1→N∗​(xN−1​)=uN−1​min​[JN→N∗​(f(xN−1​,uN−1​))+g(xN−1​,uN−1​)]

Optimal control: The optimal control uN−1∗u_{N-1}^*uN−1∗​ satisfies:

∂JN−1→N(xN−1,uN−1)∂uN−1=0\frac{\partial J_{N-1 \to N}(x_{N-1}, u_{N-1})}{\partial u_{N-1}} = 0∂uN−1​∂JN−1→N​(xN−1​,uN−1​)​=0

Stage N-2: Two Steps to Go (k=N−2→Nk = N-2 \to Nk=N−2→N)​

For the third-to-last stage:

JN−2→N(xN,xN−1,xN−2,uN−1,uN−2)=h(xN)+g(xN−1,uN−1)+g(xN−2,uN−2)\begin{aligned} &J_{N-2 \to N}(x_N, x_{N-1}, x_{N-2}, u_{N-1}, u_{N-2}) \\ &= h(x_N) + g(x_{N-1}, u_{N-1}) + g(x_{N-2}, u_{N-2}) \end{aligned}​JN−2→N​(xN​,xN−1​,xN−2​,uN−1​,uN−2​)=h(xN​)+g(xN−1​,uN−1​)+g(xN−2​,uN−2​)​

This can be rewritten as:

JN−2→N(xN−2,uN−1,uN−2)=JN−1→N(xN−1,uN−1)+g(xN−2,uN−2)\begin{aligned} &J_{N-2 \to N}(x_{N-2}, u_{N-1}, u_{N-2}) \\ &= J_{N-1 \to N}(x_{N-1}, u_{N-1}) + g(x_{N-2}, u_{N-2}) \end{aligned}​JN−2→N​(xN−2​,uN−1​,uN−2​)=JN−1→N​(xN−1​,uN−1​)+g(xN−2​,uN−2​)​

Key insight from Bellman's principle: Whatever the initial states and controls before stage N−2N-2N−2, the remaining control sequence from N−2N-2N−2 to NNN must be optimal. Therefore, uN−1∗u_{N-1}^*uN−1∗​ is already determined from the previous stage.

JN−2→N∗(xN−2)=min⁡uN−2[JN−1→N∗(f(xN−2,uN−2))+g(xN−2,uN−2)]\begin{aligned} &J_{N-2 \to N}^*(x_{N-2}) \\ &= \min_{u_{N-2}} \left[ J_{N-1 \to N}^*(f(x_{N-2}, u_{N-2})) + g(x_{N-2}, u_{N-2}) \right] \end{aligned}​JN−2→N∗​(xN−2​)=uN−2​min​[JN−1→N∗​(f(xN−2​,uN−2​))+g(xN−2​,uN−2​)]​

Optimal control:

∂JN−2→N(xN−2,uN−2)∂uN−2=0⇒uN−2∗\frac{\partial J_{N-2 \to N}(x_{N-2}, u_{N-2})}{\partial u_{N-2}} = 0 \Rightarrow u_{N-2}^*∂uN−2​∂JN−2→N​(xN−2​,uN−2​)​=0⇒uN−2∗​

General Bellman Recursion​

Bellman Equation​

For any stage kkk (where k=N−jk = N-jk=N−j for j=0,1,…,Nj = 0, 1, \ldots, Nj=0,1,…,N):

Jk∗(xk)=min⁡uk[Jk+1∗(f(xk,uk))+g(xk,uk)]\begin{aligned} J_k^*(x_k) &= \min_{u_k} \left[ J_{k+1}^*(f(x_k, u_k)) + g(x_k, u_k) \right] \end{aligned}Jk∗​(xk​)​=uk​min​[Jk+1∗​(f(xk​,uk​))+g(xk​,uk​)]​

With boundary condition:

JN∗(xN)=h(xN)J_N^*(x_N) = h(x_N)JN∗​(xN​)=h(xN​)

Optimal Control Policy​

The optimal control at each stage is:

uk∗(xk)=arg⁡min⁡uk[Jk+1∗(f(xk,uk))+g(xk,uk)]u_k^*(x_k) = \arg\min_{u_k} \left[ J_{k+1}^*(f(x_k, u_k)) + g(x_k, u_k) \right]uk∗​(xk​)=arguk​min​[Jk+1∗​(f(xk​,uk​))+g(xk​,uk​)]

Value Function Interpretation​

  • Jk∗(xk)J_k^*(x_k)Jk∗​(xk​): Value function - minimum cost-to-go from state xkx_kxk​ at time kkk
  • uk∗(xk)u_k^*(x_k)uk∗​(xk​): Policy function - optimal control as a function of current state

Properties of Dynamic Programming​

1. Principle of Optimality​

The Bellman equation embodies the principle of optimality:

An optimal policy has the property that whatever the initial state and initial decision are, the remaining decisions must constitute an optimal policy with regard to the state resulting from the first decision.

2. Backward Induction​

The solution proceeds backward in time:

  1. Start with terminal condition at k=Nk = Nk=N
  2. Solve for k=N−1,N−2,…,1,0k = N-1, N-2, \ldots, 1, 0k=N−1,N−2,…,1,0
  3. At each stage, use previously computed value function

3. Markov Property​

The optimal control depends only on the current state, not on the path taken to reach that state:

uk∗=uk∗(xk)u_k^* = u_k^*(x_k)uk∗​=uk∗​(xk​)

Algorithm Summary​

Step 1: Initialize terminal condition

JN∗(xN)=h(xN)J_N^*(x_N) = h(x_N)JN∗​(xN​)=h(xN​)

Step 2: For k=N−1,N−2,…,0k = N-1, N-2, \ldots, 0k=N−1,N−2,…,0, compute:

Jk∗(xk)=min⁡uk[Jk+1∗(f(xk,uk))+g(xk,uk)]J_k^*(x_k) = \min_{u_k} \left[ J_{k+1}^*(f(x_k, u_k)) + g(x_k, u_k) \right]Jk∗​(xk​)=uk​min​[Jk+1∗​(f(xk​,uk​))+g(xk​,uk​)] uk∗(xk)=arg⁡min⁡uk[Jk+1∗(f(xk,uk))+g(xk,uk)]u_k^*(x_k) = \arg\min_{u_k} \left[ J_{k+1}^*(f(x_k, u_k)) + g(x_k, u_k) \right]uk∗​(xk​)=arguk​min​[Jk+1∗​(f(xk​,uk​))+g(xk​,uk​)]

Step 3: Forward simulation using optimal policy Starting from x0x_0x0​, apply:

uk=uk∗(xk),xk+1=f(xk,uk∗(xk))u_k = u_k^*(x_k), \quad x_{k+1} = f(x_k, u_k^*(x_k))uk​=uk∗​(xk​),xk+1​=f(xk​,uk∗​(xk​))

Examples of Cost Functions​

1. Quadratic Cost (LQR)​

System: xk+1=Axk+Bukx_{k+1} = Ax_k + Bu_kxk+1​=Axk​+Buk​

Cost:

J=xNTSxN+∑k=0N−1(xkTQxk+ukTRuk)J = x_N^T S x_N + \sum_{k=0}^{N-1} (x_k^T Q x_k + u_k^T R u_k)J=xNT​SxN​+k=0∑N−1​(xkT​Qxk​+ukT​Ruk​)

Bellman equation:

Jk∗(xk)=min⁡uk[Jk+1∗(Axk+Buk)+xkTQxk+ukTRuk]J_k^*(x_k) = \min_{u_k} \left[ J_{k+1}^*(Ax_k + Bu_k) + x_k^T Q x_k + u_k^T R u_k \right]Jk∗​(xk​)=uk​min​[Jk+1∗​(Axk​+Buk​)+xkT​Qxk​+ukT​Ruk​]

2. Minimum Time​

Cost: J=NJ = NJ=N (number of steps)

Stage cost: g(xk,uk)=1g(x_k, u_k) = 1g(xk​,uk​)=1

Terminal cost: h(xN)=0h(x_N) = 0h(xN​)=0

3. Minimum Energy​

Cost:

J=∑k=0N−1ukTRukJ = \sum_{k=0}^{N-1} u_k^T R u_kJ=k=0∑N−1​ukT​Ruk​

Stage cost: g(xk,uk)=ukTRukg(x_k, u_k) = u_k^T R u_kg(xk​,uk​)=ukT​Ruk​

Terminal cost: h(xN)=0h(x_N) = 0h(xN​)=0

Computational Considerations​

Advantages​

  1. Global optimality: Guaranteed to find global optimum for the discretized problem
  2. Systematic approach: Provides complete optimal policy
  3. Handles nonlinearity: Works for nonlinear systems
  4. Constraint handling: Can incorporate state and input constraints

Limitations​

  1. Curse of dimensionality: Computational complexity grows exponentially with state dimension
  2. Discretization: Continuous problems must be discretized
  3. Storage requirements: Must store value function over entire state space
  4. Offline computation: Value function must be computed before implementation

Extensions​

1. Stochastic Dynamic Programming​

For systems with uncertainty:

xk+1=f(xk,uk,wk)x_{k+1} = f(x_k, u_k, w_k)xk+1​=f(xk​,uk​,wk​)

The Bellman equation becomes:

Jk∗(xk)=min⁡uk[Ewk[Jk+1∗(f(xk,uk,wk))]+g(xk,uk)]J_k^*(x_k) = \min_{u_k} \left[ \mathbb{E}_{w_k}[J_{k+1}^*(f(x_k, u_k, w_k))] + g(x_k, u_k) \right]Jk∗​(xk​)=uk​min​[Ewk​​[Jk+1∗​(f(xk​,uk​,wk​))]+g(xk​,uk​)]

2. Infinite Horizon​

For N→∞N \to \inftyN→∞, the value function becomes stationary:

J∗(x)=min⁡u[J∗(f(x,u))+g(x,u)]J^*(x) = \min_u \left[ J^*(f(x, u)) + g(x, u) \right]J∗(x)=umin​[J∗(f(x,u))+g(x,u)]

3. Approximate Dynamic Programming​

Use function approximation to handle high-dimensional problems:

Jk∗(xk)≈J^k(xk,θk)J_k^*(x_k) \approx \hat{J}_k(x_k, \theta_k)Jk∗​(xk​)≈J^k​(xk​,θk​)

where θk\theta_kθk​ are parameters to be learned.

Connection to Other Methods​

1. Model Predictive Control (MPC)​

MPC applies dynamic programming over a receding horizon.

2. Reinforcement Learning​

RL algorithms like Value Iteration implement dynamic programming with unknown models.

3. Pontryagin's Maximum Principle​

For continuous-time problems, PMP provides necessary conditions that are related to the Bellman equation.
