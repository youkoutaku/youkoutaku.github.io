離散時間システム

離散時間システムは、ディジタル信号処理の基盤となる概念です。アナログ信号をディジタル化して処理するための理論的基礎から、実際のシステム設計まで幅広くカバーします。

サンプリング定理（標本化定理）​

ナイキスト・シャノンのサンプリング定理​

サンプリング定理または標本化定理とも呼ばれ、シャノンによって確立されました。

サンプリング定理の要求

信号成分の最大周波数の2倍より高いサンプリング周波数を取るべきです。

信号 x(t)x(t)x(t) の成分の最大角周波数 ωM\omega_MωM​ が以下の条件を満たすとき、サンプリング定理を満たします：

ωM<ωs2=πT\omega_M < \frac{\omega_s}{2} = \frac{\pi}{T}ωM​<2ωs​​=Tπ​

すなわち： ωs>2ωM\omega_s > 2\omega_Mωs​>2ωM​

ここで：

  • ωs\omega_sωs​：サンプリング角周波数
  • TTT：サンプリング周期
  • ωM\omega_MωM​：信号の最大角周波数
ナイキスト周波数

ωs/2\omega_s/2ωs​/2 をナイキスト周波数と呼びます。これを超える周波数成分はエイリアシング（折り返し雑音）を引き起こします。

インパルス応答​

連続時間システムとの類推​

連続時間システムでは：

  • 入力 u(t)=δ(t)u(t) = \delta(t)u(t)=δ(t) をラプラス変換による微分方程式に代入
  • 出力 Y(s)=H(s)Y(s) = H(s)Y(s)=H(s)（伝達関数）
  • インパルス応答 h(t)h(t)h(t) は伝達関数の逆ラプラス変換

離散時間システムのインパルス応答​

離散時間システムでは：

  • 入力 x(n)=δ(n)x(n) = \delta(n)x(n)=δ(n)（離散デルタ関数）
  • 出力 Y(z)=H(z)Y(z) = H(z)Y(z)=H(z)
  • インパルス応答 h(n)h(n)h(n) は伝達関数の逆Z変換

h(n)=Z−1[H(z)]h(n) = \mathcal{Z}^{-1}[H(z)]h(n)=Z−1[H(z)]

システムの分類：IIRとFIR​

IIR（Infinite Impulse Response）システム​

  • 無限インパルス応答
  • 伝達関数 H(z)H(z)H(z) に分母がある
  • 巡回型システム
  • フィードバックを持つ

特徴：

  • 少ない係数で急峻な特性を実現
  • 位相特性が非線形
  • 安定性の考慮が必要

FIR（Finite Impulse Response）システム​

  • 有限インパルス応答
  • 伝達関数 H(z)H(z)H(z) に分母がない
  • 非巡回型システム
  • フィードフォワードのみ

特徴：

  • 常に安定
  • 線形位相特性を実現可能
  • 設計が比較的容易
システム選択の指針
  • 急峻な特性が必要：IIRシステム
  • 線形位相が重要：FIRシステム
  • 安定性が最優先：FIRシステム

ディジタル信号処理の基本要素​

ディジタル信号処理システムの基本要素：

  1. 加算器：信号の加算
  2. 乗算器：定数との乗算
  3. 遅延素子：z−1z^{-1}z−1（1サンプル遅延）

畳み込みの物理的意味

畳み込みは、システムの記憶効果を表現します。現在の出力は、過去のすべての入力とシステムの応答の重み付き和として計算されます。

群遅延は、信号の各周波数成分がシステムを通過する際の遅延時間を表します。

群遅延の重要性

通信システムや音響システムでは、群遅延の平坦性が信号の歪みを防ぐために重要です。

離散時間システムの安定性​

連続時間との対応​

連続時間システム： G(s)→G(jω),s=jωG(s) \rightarrow G(j\omega), \quad s = j\omegaG(s)→G(jω),s=jω

離散時間システム： H(z)→H(ejωT),z=ejωTH(z) \rightarrow H(e^{j\omega T}), \quad z = e^{j\omega T}H(z)→H(ejωT),z=ejωT

s平面の虚数軸は、z平面の単位円に写像される

安定性判別​

離散時間システムの安定性は極の位置で決まります：

  • 安定：すべての極の絶対値が1より小さい（∣zi∣<1|z_i| < 1∣zi​∣<1）
  • 安定限界：極の絶対値が1に等しい（∣zi∣=1|z_i| = 1∣zi​∣=1）
  • 不安定：極の絶対値が1より大きい（∣zi∣>1|z_i| > 1∣zi​∣>1）
安定性の直感的理解
  • 単位円内：時間とともに減衰（安定）
  • 単位円上：振動が持続（安定限界）
  • 単位円外：時間とともに発散（不安定）

まとめ​

離散時間システムは、ディジタル信号処理の基礎となる重要な概念です：

  1. サンプリング定理：アナログ信号のディジタル化における基本原理
  2. 伝達関数：システムの入出力関係の数学的表現
  3. インパルス応答：システムの特性を表す基本的な応答
  4. 安定性：実用的なシステム設計における重要な考慮事項
